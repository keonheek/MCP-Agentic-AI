# /diagram — Business Flowchart Generator

Generate a business-style Mermaid flowchart from prose, a topic, or a project description. Saves directly to `references/diagrams/` following the established 5-principle style guide.

## Usage

```
/diagram <topic or description>
/diagram "SDIC weekly session flow — what happens in each 2-hour block"
/diagram "YouTube Short pipeline from scrape to publish"
/diagram projects/youtube-biz/README.md
```

## What it does

Loads and executes the `diagram` skill.
