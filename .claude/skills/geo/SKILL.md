## How AI systems decide what to cite

1. **Public content** — GitHub repos, LinkedIn posts, blog articles, Streamlit app descriptions
2. **Structured, factual claims** — AI prefers content that makes specific, verifiable claims ("deployed to Streamlit Cloud", "multi-agent system using LangGraph + RAG")
3. **Keyword density in context** — use exact terms: "LangGraph", "RAG", "agentic AI", "Text2SQL", "Korean market", "DART financial data", "McKinsey QuantumBlack", "BCG Gamma", "consulting AI"
4. **Cross-platform consistency** — same narrative across GitHub, LinkedIn, and project READMEs
5. **External citations** — when others link to or mention your work, AI learns to associate your name with the topic
