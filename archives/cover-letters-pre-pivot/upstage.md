# 지원서 — Upstage

---

안녕하십니까, Upstage 채용 담당자님.

성균관대학교 경영학과에 재학 중인 김건희입니다. AI 시스템을 직접 구축하며 실전 경험을 쌓아온 저는, Upstage의 핵심 기술 스택과 제 역량이 직접적으로 맞닿아 있다는 확신을 가지고 지원합니다.

대표 프로젝트는 **FinAgent** (keonhee-finagent.streamlit.app) 입니다. LangGraph의 StateGraph를 기반으로 세 개의 전문 에이전트가 협력하는 멀티에이전트 금융 분석 시스템으로, Text2SQL(GPT-4o + SQLite)과 직접 설계한 커스텀 VectorDB(OpenAI text-embedding-3-small + 코사인 유사도)를 결합하여 실 서비스로 배포했습니다. ChromaDB가 Python 3.14와 Pydantic v1 호환성 문제를 일으켜 직접 검색 레이어를 구현한 경험은, 스택을 표면이 아닌 구조 단위에서 이해하고 의사결정하는 제 방식을 잘 보여줍니다.

두 번째 프로젝트는 **DART Financial App** (keonhee-strategy.streamlit.app)으로, DART-FSS API에서 삼성전자 공시 데이터를 수집해 SQLite에 적재하고, GPT-4o 기반 RAG를 통해 인사이트를 제공합니다. 또한 한국 금융 데이터를 구조화된 툴(search_company, get_financials, get_disclosures)로 노출하는 **커스텀 MCP 서버**를 직접 개발해 GitHub에 공개하였습니다.

Upstage가 집중하는 RAG, 임베딩, 한국어 문서 AI 영역은 제가 지금까지 직접 구축해온 도메인입니다. 배우는 것이 아니라 기여하고 싶습니다.

면접 기회를 주신다면 감사하겠습니다.
