# SDC 리소스 1: PPT 제작 가이드

---

## 핵심 원칙

**슬라이드 1장 = 메시지 1개**

- 한 슬라이드에 메시지가 2개 이상이면 분리하세요
- 슬라이드 제목 자체가 핵심 메시지여야 합니다
  - X: "시장 분석 결과"
  - O: "국내 IT 컨설팅 시장은 연 12% 성장 중"

---

## 슬라이드 구조

| 요소 | 규칙 |
|------|------|
| 제목 | 인사이트를 담은 문장 (명사형 X, 문장형 O) |
| 본문 | 3줄 이내. 부연 설명이 아니라 근거 |
| 시각 자료 | 차트, 다이어그램, 아이콘 — 텍스트 대체용 |
| 출처 | 슬라이드 하단 작은 글씨로 표기 |

---

## 디자인 기본

- **폰트 크기**: 제목 28pt 이상, 본문 18pt 이상, 출처 10pt
- **폰트 종류**: 프레젠테이션용 — Pretendard, Noto Sans KR, 맑은 고딕 중 택 1 (통일)
- **색상**: 메인 컬러 1개 + 보조 컬러 1개 + 회색. 3가지 이상 쓰지 않기
- **여백**: 슬라이드 가장자리 2cm 이상 비워두기
- **정렬**: 좌측 정렬 기본. 가운데 정렬은 제목/표지에만

## PPT 단축키 (생산성)

| 단축키 | 기능 |
|--------|------|
| Ctrl + D | 개체/슬라이드 복제 |
| Ctrl + Shift + C / V | 서식 복사 / 붙여넣기 |
| Ctrl + G | 개체 그룹화 |
| Ctrl + Shift + G | 그룹 해제 |
| Alt + F5 | 발표자 보기 모드 |
| Tab | 개체 순환 선택 |
| Ctrl + [ / ] | 폰트 크기 조절 |
| F5 | 슬라이드 쇼 시작 |

---

## 흔한 실수

1. **텍스트 벽** — 슬라이드에 문단을 그대로 옮기는 것. 키워드 + 시각화로 대체
2. **애니메이션 남용** — 전환 효과는 "없음" 또는 "페이드" 하나만. 날아오는 효과 금지
3. **불필요한 장식** — 클립아트, 워드아트, 그라데이션 배경 금지
4. **비율 불일치** — 이미지를 늘려서 비율이 깨진 상태로 사용하지 않기
5. **출처 미표기** — 데이터나 인용은 반드시 출처 표기

## 5-7장 슬라이드 템플릿

이번 과제 기준:

| 슬라이드 | 내용 |
|----------|------|
| 1 | 표지: 팀명, 주제, 날짜 |
| 2 | 컨설팅이란? — 핵심 정의 + 주요 역할 |
| 3 | IT 컨설팅이란? — 일반 컨설팅과의 차이점 |
| 4-5 | 리서치에서 얻은 핵심 인사이트 (각 1장씩) |
| 6 | SDC에 적용할 수 있는 점 — 구체적 행동 방안 |
| 7 | 요약 + 다음 단계 |
