# 데모 스크립트 (영범 또는 직접 영업용)

## 목표

리드가 "무료 GEO 스캔 해줘"라고 했을 때, 24시간 내 PDF 리포트 + 유료 서비스 피치를 전달한다.

---

## Step 1: 무료 스캔 요청 접수

**카카오톡 수신 메시지 예:**
> "무료 진단 해주신다고 하셨죠? 저희 브랜드 한번 봐주세요."

**즉시 답장:**
```
네! 브랜드명이랑 웹사이트 URL 알려주시면 24시간 내로 AI 검색 가시성 진단 리포트 보내드릴게요 :)
```

---

## Step 2: 스캔 실행

터미널에서:
```bash
python demo/run_demo.py --brand [브랜드명] --live
```

출력물:
- `demo/[날짜]-[브랜드]-geo-audit.md` (PDF 없을 경우 Markdown)
- GEO 점수 + 개선 권장사항

---

## Step 3: 리포트 전달 + 피치

카카오톡 메시지 (카피 앤 페이스트):

```
안녕하세요 :) [브랜드명] GEO 진단 결과 나왔어요!

AI 검색 가시성 점수: [점수]/100 ([평가])
ChatGPT/Perplexity에서 브랜드 언급 빈도: [X]/10회

주요 개선 포인트:
1. [개선사항 1]
2. [개선사항 2]

리포트 첨부드려요. (파일 첨부)

저희 GEO/SEO 블로그 서비스 이용하시면 3개월 내 점수 20점 이상 올리는 걸 목표로 합니다.
가격은 셋업 50만원 + 월 30만원 (4편/월)이에요.

관심 있으시면 말씀해주세요 :)
```

---

## Step 4: 팔로우업 (3일 후 무응답 시)

```
안녕하세요, 혹시 리포트 확인하셨나요? :)
궁금한 점 있으시면 언제든 편하게 물어봐 주세요.
```

---

## 자주 나오는 질문 (Q&A)

**Q: AI 검색 최적화가 실제로 매출에 도움이 되나요?**
> ChatGPT, Perplexity 사용자들이 구매 전 AI에게 제품 추천을 물어보는 비율이 빠르게 증가하고 있어요. AI가 브랜드를 인용하면 트래픽과 신뢰도가 올라갑니다.

**Q: 네이버 블로그 직접 운영하면 안 되나요?**
> 직접 운영하셔도 되지만 GEO 최적화 구조(FAQ, JSON-LD 스키마)를 모르면 AI 인용이 잘 안 됩니다. 저희가 그 구조를 매 포스트에 적용해드려요.

**Q: 결과 보장이 되나요?**
> 점수 상승은 보장하지만 매출 상승은 보장하기 어려워요. 대신 3개월 후 GEO 점수 20점 이상 상승을 못하면 다음 달 무료로 서비스 제공드립니다.
