# 정부지원사업 신청서 AI 초안 (Government-Grant Application Draft Agent)

모두의창업 + 소상공인 진흥공단 + K-스타트업 신청서를 RAG + GPT-4o로 자동 초안 생성하는 한국 창업자 지원 통합자. 슬라이드가 아니라 작동하는 시스템을 인도합니다.

**사업계획서 10개 공통 질문을 3시간 안에 초안 완성. 컨설팅 업체 ₩3M 대비 ₩500K-1M으로 동일 결과.**

운영사: 1stmover | 문의: keonhee3337@gmail.com

## 무엇을 해결하나요?

예비/초기 창업자들이 공통으로 겪는 문제:

- 모두의창업, 소상공인 진흥공단, K-스타트업마다 동일한 질문을 반복 작성 (Q1 동기, Q2 시장, Q3 차별성)
- 신청서 1개에 20시간 이상 소요, 탈락률 80%+ 이상
- 컨설팅 업체는 ₩3M 이상 요구
- 지원사업 공고를 놓치는 경우가 많음

---

## 작동 방식

```
클라이언트 사업 정보 입력 (Tally 폼)
        |
RAG가 과거 합격 신청서에서 유사 사례 검색
        |
GPT-4o가 공고별 양식에 맞게 초안 생성
        |
DOCX 파일 자동 생성 (기관별 양식)
        |
사업 예산표 자동 생성 (스프레드시트)
        |
1stmover 검토 후 클라이언트 전달 (24시간)
```
