# Demo Script: 정부지원사업 신청서 AI 초안

대상: 예비/초기 창업자, 소상공인
소요 시간: 10분
목표: STANDARD 계약 (₩800K) 또는 PREMIUM 계약 (₩1M)

---

## 사전 준비

- [ ] demo_grant.py 실행 확인
- [ ] 샘플 공고 PDF 준비 (모두의창업 2026 최신 공고)
- [ ] 샘플 합격 신청서 3개 RAG 인덱싱 완료
- [ ] DOCX 출력 테스트 완료

---

## 오프닝 (1분)

"안녕하세요. 지원사업 신청서를 준비 중이신가요? 오늘 10분 안에 AI가 어떻게 초안을 만들어드리는지 직접 보여드리겠습니다."

"혹시 지금 어떤 지원사업을 신청하려고 계세요?" (답변 듣기)

"컨설팅 업체에 맡기면 보통 얼마 드리던가요?" (답변 듣기)

---

## 데모 단계 1: 공고 분석 (3분)

```bash
python demo/demo_grant.py --program "모두의창업" --mode analyze
```

"이게 이번 공고의 평가 기준입니다. Q1부터 Q9까지 각 항목이 몇 점짜리인지, 어떤 키워드를 넣어야 하는지 자동으로 분석했습니다."

---

## 데모 단계 2: 초안 생성 (5분)

```bash
python demo/demo_grant.py --program "모두의창업" --business "AI 학원 자동화" --mode draft
```

"30초 안에 Q1 동기부터 Q9 수익 계획까지 전체 초안이 나옵니다. 직접 보시죠."

(출력 확인 후)

"이건 과거에 합격한 신청서 3개를 참고해서 만든 겁니다. 말투, 구조, 키워드 배치가 심사위원이 선호하는 패턴으로 정리됩니다."

---

## 데모 단계 3: DOCX 출력 (2분)

```bash
python demo/demo_grant.py --program "모두의창업" --mode export
```

"이게 실제로 제출 가능한 DOCX 파일입니다. 기관 양식에 맞춰서 바로 업로드하면 됩니다."

---

## 요금 설명 (2분)

"컨설팅 업체 ₩3M 대신 ₩800K에 같은 결과물을 48시간 이내 드립니다."

"BASIC ₩500K는 초안만, STANDARD ₩800K는 초안 + 예산표 + 수정 1회, PREMIUM ₩1M은 여기에 최종 검토까지 포함됩니다."
