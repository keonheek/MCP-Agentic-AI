# 학원 알림톡 AI 비서 (Hagwon Parent-Comms Agent)

카카오 알림톡 + 카카오 알림톡 API + GPT-4o + 학원 말투 Notion 시트를 n8n self-host로 묶은 학원 학부모 소통 통합자. 슬라이드가 아니라 작동하는 시스템을 인도합니다.

**학부모 알림톡을 90초 이내 자동 생성. ₩100K/월 retainer = ops 4시간 = 월 1회 응대 누락 방지시 손익분기.**

## 무엇을 해결하나요?

학원 원장님들이 공통으로 겪는 문제:

- 수업 끝나면 학부모 알림톡(출결, 숙제, 진도, 상담)을 매번 손으로 작성
- EduPie, 알리미, 해듀 등 EMR 시스템과 카카오 알림톡 사이를 수동 복사
- 원장 혼자 10명 이상 학부모에게 개별 메시지를 보내는 데 2시간 소요
- 말투가 일관되지 않아 학부모 신뢰 저하

---

## 작동 방식

```
EMR/해듀/알리미에서 출결/진도 데이터 입력
        |
n8n webhook이 이벤트 감지
        |
GPT-4o가 학원 말투 시트 참조해 알림톡 초안 생성
        |
카카오 알림톡 자동 발송 (90초 이내)
        |
Notion 대시보드에 발송 이력 기록
        |
판단 불가 케이스 -> 원장님 즉시 알림
```
