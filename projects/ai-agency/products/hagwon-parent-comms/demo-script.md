# Demo Script: 학원 알림톡 AI 비서

대상: 학원 원장 (1인~5인 운영, 영어/수학/피아노 학원)
소요 시간: 15분
목표: DEMO 계약 (₩500K) 또는 STANDARD 계약 (₩500K 설치 + ₩150K/월)

---

## 사전 준비

- [ ] ngrok 터널 활성화 확인 (`ngrok http 5000`)
- [ ] demo_hagwon.py 실행 확인
- [ ] 테스트용 카카오 채널 준비 (원장 폰으로 직접 받아볼 수 있게)
- [ ] Notion 대시보드 링크 준비

---

## 오프닝 (2분)

"원장님, 안녕하세요. 저는 1stmover의 김건희입니다. 오늘은 원장님이 매일 하시는 알림톡 업무를 어떻게 자동화할 수 있는지 15분 안에 직접 보여드리겠습니다."

"혹시 하루에 알림톡 작성에 얼마나 시간을 쓰시나요?" (답변 듣기)

"저희 데이터로는 학원 원장님들이 평균 1-2시간을 쓰고 계십니다. 지금 보여드릴 시스템은 그 시간을 90% 줄여드립니다."

---

## 데모 단계 1: 출결 알림톡 자동 생성 (5분)

"제가 지금 출결 정보를 입력할게요."

```bash
python demo/demo_hagwon.py --scenario attendance --student "김민준" --status "결석" --reason "발열"
```

"90초 안에 원장님 폰으로 알림톡이 옵니다. 직접 확인해보세요."

(알림톡 수신 확인 후)

"말투가 자연스럽죠? 이건 원장님이 직접 작성하신 30개 예시 문장을 AI가 학습해서 생성한 겁니다. '우리 학원 말투'로 나오는 거예요."

---

## 데모 단계 2: 숙제 미제출 알림 (3분)

"이번엔 숙제 미제출 알림입니다."

```bash
python demo/demo_hagwon.py --scenario homework --student "박서연" --subject "수학" --missing_count 3
```

"3회 연속 미제출이면 자동으로 톤이 바뀝니다. 처음엔 부드럽게, 반복되면 좀 더 진지하게. 이 규칙도 원장님이 직접 설정하실 수 있습니다."

---

## 데모 단계 3: Notion 대시보드 (2분)

Notion 링크 열기

"모든 발송 이력이 여기 자동으로 기록됩니다. 언제 보냈는지, 읽었는지, 에스컬레이션이 필요한 건지. 원장님이 직접 관리 안 하셔도 됩니다."

---

## 요금 설명 (2분)

"요금은 두 가지입니다."

"첫 번째, DEMO: ₩500K 한 번. 1개 시나리오 5일 안에 납품해드립니다. 직접 써보시고 결정하시면 됩니다."

"두 번째, STANDARD: 설치 ₩500K + 월 ₩150K. 출결/숙제/진도/상담 전체 자동화입니다."

"₩150K/월은 원장님 시간 1시간 인건비입니다. 하루 1시간 절약이면 첫 달부터 본전입니다."

## 예상 반론 대응

**"카카오 알림톡이 스팸처럼 보이지 않을까요?"**
"알림톡은 카카오가 공식 인증한 비즈니스 메시지입니다. 문자 스팸과 다릅니다. 학부모님이 카카오톡 채널을 통해 받으시는 공식 메시지예요."

**"말투가 이상하면 어떡하나요?"**
"처음 일주일은 원장님이 모든 발송 내용을 Notion에서 확인하시고 피드백 주시면 됩니다. 말투 시트를 수정하면 즉시 반영됩니다."

**"EMR이랑 연동이 안 되면요?"**
"해듀, 알리미가 안 되면 CSV 파일 업로드 방식으로도 연동됩니다. 30초면 됩니다."
