# SLA: 학원 알림톡 AI 비서

운영사: 1stmover | 최종 업데이트: 2026-05-12

---

## 가용성 목표

| 항목 | 목표 |
|---|---|
| 월간 시스템 가용성 | 99.0% (월 7.2시간 허용 다운타임) |
| 알림톡 발송 성공률 | 98% 이상 |
| 에스컬레이션 응답 (원장 알림) | 5분 이내 |
| 장애 감지 후 복구 | 4시간 이내 (영업일), 12시간 (주말/공휴일) |

---

## 지원 응답 시간

| 우선순위 | 정의 | 응답 시간 | 해결 목표 |
|---|---|---|---|
| P1 (긴급) | 전체 시스템 중단, 발송 0% | 1시간 이내 | 4시간 이내 복구 |
| P2 (높음) | 특정 시나리오 발송 실패 | 4시간 이내 | 24시간 이내 복구 |
| P3 (보통) | 말투 오류, 포맷 문제 | 1영업일 이내 | 3영업일 이내 수정 |
| P4 (낮음) | 기능 개선 요청 | 3영업일 이내 응답 | 다음 배포 포함 여부 협의 |

---

## 포함 서비스

- n8n 워크플로우 모니터링 (24/7 자동)
- Kakao Alimtalk API 상태 모니터링
- 월간 성과 리포트 (발송 건수, 성공률, 절약 시간)
- 말투 시트 업데이트 (월 1회, 원장 요청 기준)
- 시나리오 소규모 수정 (월 2회까지 무료)

---

## 포함되지 않는 항목

- EMR 시스템 자체 장애 (클라이언트 측 이슈)
- Kakao 정책 변경으로 인한 채널 차단
- 신규 시나리오 추가 (별도 견적)
- 학원 이전 또는 브랜드 변경에 따른 전면 재설정

---

## 페널티 조항

| 가용성 | 페널티 |
|---|---|
| 99.0% 이상 | 없음 |
| 98.0%-98.9% | 월 요금 10% 크레딧 |
| 97.0%-97.9% | 월 요금 20% 크레딧 |
| 97.0% 미만 | 월 요금 전액 크레딧 |

---

## 데이터 및 개인정보

- 학부모 연락처 및 학생 정보: 암호화 저장, PIPA 준수
- 발송 이력: 6개월 보관 후 삭제 (원장 요청 시 즉시 삭제)
- 데이터 소유권: 학원에 귀속. 계약 종료 시 30일 이내 전달 또는 삭제

---

## 계약 종료 조건

- 1개월 전 서면(카카오톡/이메일) 통보
- 종료 후 30일: 데이터 내보내기 지원
- 조기 종료 (3개월 이전): 잔여 설치비 환불 없음, 월 요금만 정산
