# 학원 수강료 결제 자동화 (Hagwon Tuition Auto-Billing)

Toss Payments billing key + 카카오 알림톡 + Supabase를 n8n self-host로 묶은 학원 결제 자동화 통합자. 슬라이드가 아니라 작동하는 시스템을 인도합니다.

**매달 수강료 입금 확인과 미납 독촉을 자동 처리. ₩70K/월 retainer = 원장 1시간 = 월 1건 미납 방지시 손익분기.**

운영사: 1stmover | 문의: keonhee3337@gmail.com

## 무엇을 해결하나요?

학원 원장님들이 공통으로 겪는 문제:

- 매달 초 수강료 입금 여부를 통장 내역과 학생 명단을 대조하며 수동 확인
- 미납 학부모에게 독촉 연락하기가 불편하고 감정 소모
- 학생 수가 늘수록 관리 부하 급증
- 사이다페이 사용 중이지만 비용이 비쌈 (거래당 수수료)

---

## 작동 방식

```
월 결제일 도래 (자동 감지)
        |
Toss Payments billing key로 자동 결제 시도
        |
결제 성공: 학부모에게 영수증 알림톡 자동 발송
결제 실패: 원장에게 즉시 알림 + 학부모에게 재시도 안내 알림톡
        |
Supabase에 결제 이력 자동 기록
        |
D+3 미납 시: 자동 독촉 알림톡 (원장이 직접 쓰지 않아도 됨)
        |
월말 결제 현황 리포트 자동 생성
```
