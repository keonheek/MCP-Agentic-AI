# 동의 화면 UX 문구 가이드

PIPA 제22조 준수 동의 화면 한국어 표준 문구

---

## 1. 주문/회원가입 기본 동의

### 필수 동의 (거부 시 서비스 이용 불가)

```
[필수] 개인정보 수집/이용 동의

수집 항목: 이름, 이메일, 전화번호, 배송 주소
수집 목적: 주문 처리, 배송, 고객 상담
보유 기간: 전자상거래법에 따라 5년

[자세히 보기]

□ 동의합니다 (필수)
```

### 선택 동의 1: 마케팅

```
[선택] 마케팅 정보 수신 동의

수집 항목: 이름, 이메일, 전화번호
수집 목적: 할인 정보, 신제품 안내, 이벤트 안내
보유 기간: 동의 철회 시까지

※ 동의하지 않아도 주문 및 서비스 이용에 불이익이 없습니다.

□ 동의합니다 (선택)
```

### 선택 동의 2: AI 처리 (2026 필수 추가 항목)

```
[선택] AI 자동화 처리 동의

1stmover AI Agency의 AI 시스템(Claude API, Anthropic PBC, 미국)이
귀하의 상담 내용과 구매 이력을 분석하여 자동 응대, 개인화 추천,
마케팅 우선순위 분류에 활용합니다.

수집 항목: 상담 내용, 구매 이력, 행동 데이터
목적: AI 기반 개인화 서비스
국외 이전: Anthropic PBC (미국), Make.com (EU)
이전 목적: AI 처리 및 자동화 워크플로우

※ 동의하지 않으면 AI 기반 개인화 서비스를 이용할 수 없습니다.
※ 동의를 철회하면 AI 처리가 즉시 중단됩니다.

□ 동의합니다 (선택)
```

### 선택 동의 3: 민감정보 (피부 데이터 수집 브랜드 필수)

```
[선택] 민감정보 수집/이용 동의 (개인정보 보호법 제23조)

수집 항목: 피부 타입, 피부 상태, 알레르기 성분, 피부 민감도
수집 목적: 피부 맞춤 제품 추천, 성분 안전성 분석
보유 기간: 동의 철회 시까지
보안: AES-256 암호화 저장

※ 민감정보는 피부 추천 목적으로만 사용되며 제3자에게 제공되지 않습니다.
※ 동의하지 않아도 일반 제품 구매는 가능합니다.

□ 동의합니다 (선택)
```

## 2. 카카오 채널/챗봇 동의 문구

첫 메시지 자동 발송용:

```
안녕하세요! [브랜드명] AI 상담 채널입니다.

원활한 상담을 위해 아래 정보 수집에 동의해 주세요.

[수집 항목] 이름, 상담 내용
[이용 목적] AI 자동 상담, 제품 추천
[보유 기간] 상담 종료 후 1년
[국외 이전] Anthropic Claude API (미국) 활용

✅ 동의하고 상담 시작하기
❌ 동의 없이 상담하기 (일부 기능 제한)
```

## 3. 동의 철회 안내 문구 (PIPA 제37조)

```
개인정보 처리 동의는 언제든지 철회할 수 있습니다.

철회 방법:
- 마이페이지 > 개인정보 관리 > 동의 관리
- 이메일: [담당자 이메일]
- 전화: [고객센터 번호]

동의 철회 시 AI 처리 및 마케팅 발송이 즉시 중단됩니다.
기존 주문 정보는 전자상거래법에 따라 5년간 보관됩니다.
```

## 4. 자동화 결정 설명 요구 안내 (PIPA 제37조의2, 2026 신설)

```
[AI 결정 설명 요구권 안내]

당사 AI 시스템이 귀하의 마케팅 등급, 제품 추천, 상담 우선순위를
자동으로 결정할 수 있습니다.

이 결정에 이의가 있으시면 설명을 요구할 수 있습니다:
- 요청 방법: 이메일 [담당자 이메일] (제목: "AI 결정 설명 요구")
- 응답 기한: 요청 후 10 영업일 이내

담당자의 직접 검토를 요청하실 수도 있습니다.
```

## 5. 체크리스트: 개발 적용 전 확인사항

- [ ] 필수/선택 동의 분리 (하나의 체크박스에 묶지 않기)
- [ ] 사전 체크 없음 (체크박스는 항상 비어 있는 상태로 시작)
- [ ] 동의 거부 시 불이익 명시
- [ ] 동의 일시, IP 주소, 버전 서버 기록
- [ ] 철회 경로가 동의 경로만큼 쉬울 것 (PIPA 제37조)
- [ ] 국외 이전 대상 서비스 명시 (Anthropic, Make.com 등)
- [ ] 민감정보 별도 동의 화면
