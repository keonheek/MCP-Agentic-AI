# 개인정보 보호법 2026 개정 핵심 요약

작성 기준: 2026년 개정안 (2026.09.11. 발효) + 2026.02.12. 시행령 개정

대상 독자: 한국 화장품/건강기능식품 D2C 브랜드 대표 및 마케팅 담당자

---

## 1. 왜 지금 중요한가

| 구분 | 기존 (2023) | 2026 개정 |
|---|---|---|
| AI 처리 | 규정 없음 | AI 사용 시 처리 명세서 의무 |
| 침해 신고 기한 | 5 영업일 이내 | **72시간 이내** |
| 과징금 한도 | 매출액 3% | **매출액 10%** (고위험 침해) |
| 국외 이전 | 고지 권고 | **명시적 동의 필수** |
| 자동화 결정 | 규정 없음 | 설명 요구권 신설 (제37조의2) |

## 2. 조문별 핵심 의무

### 제15조 (수집, 이용)
개인정보를 수집할 때 반드시 정보주체의 동의를 받아야 합니다.
수집 목적, 항목, 보유 기간을 명확히 고지해야 합니다.
**화장품 브랜드 체크포인트:** 카카오 채널, 스마트스토어 주문 시 동의 화면 필수.

### 제17조 (제3자 제공)
마케팅사, 물류사, AI 서비스 업체에 개인정보를 제공할 때마다 별도 동의가 필요합니다.
**화장품 브랜드 체크포인트:** Make.com, Anthropic API, 배송 대행사는 모두 제3자입니다.

### 제22조 (동의 방법)
동의는 "명확하게 표시된 행위"여야 합니다. 사전 체크박스 불가.
AI 처리 목적의 동의는 마케팅 수신 동의와 별도로 받아야 합니다.

### 제23조 (민감정보)
피부 상태, 알레르기, 건강 정보는 민감정보입니다.
수집 시 별도 동의 + 암호화 저장 필수.

### 제24조 (고유식별정보)
주민등록번호, 여권번호 등은 법령에 근거 없이 수집 불가.
불가피하게 처리 시 암호화 필수.

### 제26조 (처리위탁)
위탁 업체 목록과 업무 내용을 개인정보처리방침에 공개해야 합니다.
위탁 계약서(DPA) 보관 의무.

### 제28조의2 (국외 이전) [2026 주요 변경]
해외 AI API, 자동화 툴, 클라우드 서비스로 데이터를 전송하는 경우:
1. 이전받는 자, 이전 국가, 이전 목적, 이전 항목 고지
2. 정보주체의 명시적 사전 동의 취득
3. 개인정보처리방침에 국외 이전 현황 공개

**대상 서비스 예시:** Anthropic Claude API (미국), OpenAI GPT API (미국),
Make.com (EU), Google Workspace (미국), Mailchimp (미국)

### 제29조 (안전조치)
데이터 규모에 따른 안전조치 의무:
- 1만 명 이상: 침입탐지 시스템, 취약점 점검 의무
- 개인정보보호 책임자(CPO) 지정 의무 (매출 1억 이상 or 1,000명 이상 처리)
암호화: 주민번호, 비밀번호, 금융정보, 바이오정보는 AES-256 이상 필수.

### 제34조 (침해 신고) [2026 주요 변경]
개인정보 유출 인지 후 72시간 이내 개인정보보호위원회(PIPC)에 신고.
정보주체에게도 "지체 없이" 통지.
1,000명 이상: 홈페이지 공지 추가 의무.
**신고처:** privacy.go.kr

### 제35조 (열람청구)
정보주체는 언제든지 자신의 개인정보 열람을 요구할 수 있습니다.
10 영업일 이내 응답 의무. 미응답 시 과태료 최대 1천만 원.

### 제37조의2 (자동화 결정 설명) [2026 신설]
AI가 정보주체에게 중요한 영향을 미치는 결정을 내린 경우
(예: 마케팅 등급 분류, 신용 평가, 개인화 추천),
정보주체는 해당 결정의 근거 설명을 요구할 수 있습니다.
10 영업일 이내 응답 의무.

## 3. 화장품 D2C 브랜드가 수집하는 데이터와 해당 조문

| 데이터 항목 | 해당 조문 | 위반 시 제재 |
|---|---|---|
| 이름, 전화번호, 이메일 | 제15조, 제22조 | 과태료 최대 3천만 원 |
| 주소 (배송지) | 제15조, 제17조 | 과태료 최대 3천만 원 |
| 피부 타입/상태, 알레르기 | 제23조 (민감정보) | 과태료 최대 5천만 원 |
| 결제 정보, 카드번호 | 제23조, 제29조 | 과징금 매출 10% |
| AI 챗봇 대화 내용 | 제15조, 제28조의2 (국외 이전) | 과태료 최대 3천만 원 |
| 마케팅 수신 이력 | 제22조 | 과태료 최대 3천만 원 |

## 4. 과징금/과태료 요약 (2026 기준)

| 위반 유형 | 제재 수준 |
|---|---|
| 동의 없는 수집/이용 | 과태료 최대 3천만 원 |
| 민감정보 무단 수집 | 과태료 최대 5천만 원 |
| 침해 신고 72시간 위반 | 과태료 최대 3천만 원 |
| 국외 이전 동의 없음 | 과태료 최대 3천만 원 |
| 고위험 침해 (건강, 금융, 주민번호) | **과징금 매출액 최대 10%** |
| 안전조치 미비로 인한 유출 | 과징금 매출액 최대 3% |

## 5. D-day 카운트다운

2026년 9월 11일 이전에 다음 항목을 완료해야 합니다:

- [ ] 개인정보처리방침 업데이트 (국외 이전 현황 포함)
- [ ] 동의 화면 재설계 (AI 처리 목적 별도 동의)
- [ ] 위탁 업체 목록 정비 및 DPA 체결
- [ ] 침해 대응 절차서 수립 (72시간 신고 프로세스)
- [ ] 민감정보 암호화 적용 (피부상태, 알레르기)
- [ ] 자동화 결정 설명 프로세스 구축
