# 데모 스크립트

대상: 잠재 클라이언트 (화장품 D2C 브랜드 대표)
소요 시간: 10-15분
채널: 화상 미팅 또는 대면

---

## 오프닝 (1분)

"안녕하세요, 저는 AI 에이전시를 운영하는 김건희입니다.
오늘은 [클라이언트 쇼핑몰명]에서 매일 반복하시는 수작업들을 자동화하는 방법을 보여드리려고 합니다.
화면에서 실제 작동하는 것을 보여드릴게요."

---

## 1단계: 문제 공감 (2분)

"지금 이런 상황이시죠?"

- 주문이 들어오면 고객에게 수동으로 카카오톡이나 문자 보내고 계시죠?
- 하루에 [N]건이면 그것만 해도 [N x 2]분씩 쓰고 계신 겁니다.
- 재고가 떨어진 걸 판매 다 되고 나서야 아시는 경우 있으시죠?
- CS 인입되면 같은 질문에 매일 답하고 계시죠? (배송 언제 오나요, 교환 어떻게 하나요)

"이걸 저희가 다 자동으로 처리해드립니다."

---

## 2단계: 라이브 데모 (5분)

### 실행 명령어

```bash
cd projects/ai-agency/products/saas-integrations
python demo/run_demo.py
```

### 설명 포인트

화면에 출력되는 통합 테이블을 보여주며:

"보시는 것처럼, Cafe24에서 주문이 들어오든, 스마트스토어에서 들어오든, Shopify에서 들어오든 저희 시스템은 동일하게 처리합니다.
플랫폼마다 API 형태가 다르거든요. 저희가 그 차이를 다 흡수해드립니다."

자동화 흐름 설명:

1. "결제 완료 웹훅이 저희 서버에 도착합니다. (실시간, 30초 이내)"
2. "서명 검증으로 위변조 방지합니다."
3. "카카오 알림톡이 고객에게 자동 발송됩니다."
4. "고객이 채널톡으로 문의하면 FAQ는 자동 응답, 복잡한 건은 담당자에게 자동 연결됩니다."

---

## 3단계: 수치 제시 (2분)

"계산해 드릴게요."

- 월 주문 [N]건 x 알림 발송 3분 = 월 [N x 3]분 = [시간]시간 절감
- CS 반복 문의 자동화 시 월 [N]시간 절감
- 월 200,000원으로 이 시간을 다 돌려받으시는 겁니다.

"사람 한 명 파트타임 고용하시는 것의 [N]분의 1 가격입니다."

---

## 4단계: 플랫폼 확인 (1분)

"지금 Cafe24 쓰고 계시죠? / 스마트스토어도 같이 쓰시나요?"

"저희가 두 곳 다 연동해드릴 수 있습니다. 한 번만 설정하면 두 채널 모두 자동화됩니다."

---

## 5단계: 클로징 (2분)

"오늘부터 2주 안에 전부 세팅 완료해 드립니다."

"필요하신 게 딱 두 가지예요.
첫째, [플랫폼] 관리자 페이지에서 API 키 하나 발급해주시면 됩니다. 제가 가이드 드릴게요.
둘째, 카카오 비즈니스 계정 있으시면 그걸로 바로 시작할 수 있습니다."

"바로 시작하시겠어요? 초기 설치비 [금액]원, 이후 월 200,000원입니다."

## 자주 나오는 질문 (Q&A 대비)

**Q: 지금 Cafe24 직원이 따로 있는데 걔네들이랑 충돌 나지 않나요?**
A: 저희는 API로만 연동하기 때문에 기존 Cafe24 운영에 전혀 영향이 없습니다.

**Q: 보안은 어떻게 되나요?**
A: 모든 웹훅 수신 시 HMAC 서명 검증을 합니다. API 키는 저희 서버 환경 변수에만 저장됩니다.

**Q: 나중에 다른 플랫폼 추가하면 얼마예요?**
A: 추가 플랫폼당 초기 설치비 200,000원입니다. 월 요금은 그대로입니다.

**Q: 카카오 알림톡 심사 얼마나 걸려요?**
A: 보통 1-3 영업일입니다. 저희가 템플릿 작성부터 심사 제출까지 대행합니다.

**Q: 중간에 해지하면요?**
A: 월 단위 계약이라 다음 달부터 중단 가능합니다. 위약금 없습니다.
