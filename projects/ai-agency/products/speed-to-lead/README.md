## 무엇을 해결하나요?

화장품 D2C 브랜드 사장님들이 공통으로 겪는 문제:

- 카카오톡 문의가 오는데 바로 답변하기 어렵다
- 밤/주말에도 문의가 오는데 다음날 보면 이미 경쟁사로 넘어간다
- 같은 질문이 반복해서 온다 (가격, 성분, 재주문 방법 등)
- 문의 이력을 체계적으로 관리하기 어렵다

## 작동 방식

```
고객이 카카오톡 채널에 문의
        |
AI가 5개 카테고리로 자동 분류
(견적 / 제품문의 / 재구매 / 예약 / 기타)
        |
브랜드 맞춤형 답변 자동 생성
        |
카카오톡 자동 발송 (90초 이내)
        |
Notion 대시보드에 전체 이력 기록
        |
판단 불가 문의 -> 사장님 즉시 알림
```
