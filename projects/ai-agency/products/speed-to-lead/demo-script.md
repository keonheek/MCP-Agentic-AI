# Speed-to-Lead 라이브 데모 스크립트

_영업 통화 / 화면 공유 전용. 소요 시간: 약 20-30분._

---

## 데모 전 체크리스트 (통화 15분 전)

- [ ] Flask 서버 실행 확인: `python speed_to_lead.py`
- [ ] ngrok 터널 가동 확인
- [ ] `test-client` 슬러그로 smoke test 통과 (6/6 pass)
- [ ] Notion 대시보드 브라우저 탭 열기
- [ ] `demo/run_demo.py` 실행 준비 완료
- [ ] 폰 배터리 확인 (실문의 테스트용)

---

## 오프닝 (3분)

**목적:** 문제를 공감으로 시작한다.

> "사장님, 카카오톡 채널에 문의가 오는데 바로 답변하기 어려우신 적 많으시죠?
> 특히 밤에 오는 문의, 주말 문의, 같은 질문 반복해서 오는 것들 처리하다 보면
> 하루에 몇 시간씩 쓰게 되잖아요.
>
> 오늘은 그걸 자동으로 처리하는 시스템을 직접 보여드릴게요.
> 제가 지금 바로 실제로 작동하는 걸 보여드리겠습니다."

---

## Part 1. 자동 분류 시연 (5분)

**화면:** 터미널 + `run_demo.py` 실행

> "지금 보시는 게 실제 AI 처리 결과입니다.
> 고객이 문의를 보내면 먼저 어떤 종류의 문의인지 자동으로 분류해요."

`python demo/run_demo.py --cached` 실행 후 출력 결과 설명:

- 견적 문의 → "견적" 분류, 90% 신뢰도
- 제품문의 → "제품문의" 분류
- 재구매 → "재구매" 분류
- 예약 → "예약" 분류
- 기타 → "기타" 분류, 에스컬레이션 알림

> "신뢰도가 낮은 문의는 자동으로 사장님께 알림이 가요.
> 나머지는 AI가 바로 답변합니다."

---

## Part 2. 실시간 응답 시연 (5분)

**화면:** Notion 대시보드 (한쪽) + 터미널 (다른 쪽)

> "이번엔 실제 문의를 넣어볼게요. 제가 지금 문의를 보낼게요."

`test_webhook.py` 또는 `curl` 로 문의 전송:

```bash
curl -X POST http://localhost:5000/webhook/test-client \
  -H "Content-Type: application/json" \
  -d '{"userRequest": {"user": {"id": "demo_live"}, "utterance": "세럼 10개 도매가 가능한가요?"}}'
```

> "보내자마자 Notion 대시보드에 바로 기록이 됩니다. [Notion 화면 전환]
> 카테고리, AI 답변 내용, 신뢰도, 처리 시간이 다 남아요.
> 사장님이 하루 끝나고 보시면 오늘 어떤 문의가 얼마나 왔는지 한눈에 보여요."

---

## Part 3. 에스컬레이션 시연 (3분)

**목적:** "AI가 혼자 다 결정하는 게 아니라 나도 컨트롤할 수 있다"는 안심 제공

> "판단이 어려운 문의, 예를 들어 불만 고객이거나 애매한 문의가 오면
> AI가 직접 답변 안 하고 사장님께 바로 알림을 보냅니다."

단어 하나짜리 문의 전송 (예: "어") 후 에스컬레이션 로그 확인:

> "이렇게 AI가 혼자 답변하기 어렵다고 판단하면 사장님 폰에 알림이 가요.
> 그때 사장님이 직접 답변하시면 됩니다. 전부 자동으로 할 필요 없어요.
> 애매한 건 사람이, 반복적인 건 AI가 처리하는 구조예요."

---

## Part 4. 브랜드 맞춤화 설명 (3분)

**목적:** "우리 브랜드에 맞게 설정 가능하다"는 확신

> "지금 보신 건 테스트 브랜드로 세팅한 거고요.
> 사장님 브랜드 이름, 제품 목록, 자주 오는 질문 넣어드리면
> 답변도 사장님 브랜드 톤으로 나와요.
>
> 예를 들어 '우리 브랜드는 친근한 언니 같은 느낌'이라고 하시면
> 그 톤으로 답변이 나오도록 설정해 드립니다."

---

## Part 5. 가격 및 다음 단계 (5분)

> "오늘 데모 어떠셨어요? 실제로 운영에 바로 적용하실 수 있을 것 같으신가요?"

[긍정 반응 시]

> "저희가 지금 처음 3개 클라이언트에게만 웨지 가격으로 드리고 있어요.
> 설치비 50만 원, 월 20만 원인데, 30일 환불 보장이에요.
> 딱 한 달 써보시고 아니다 싶으면 전액 환불해 드립니다.
>
> 시작하시면 카카오톡 채널 API 설정만 10분만 도와주시면 나머지는 제가 다 세팅해 드려요.
> 빠르면 내일부터 가동 가능합니다. 어떻게 생각하세요?"

[가격 반발 시]

> "이해해요. 한 달 처음 써보시는 거라 부담되실 수 있어요.
> 30일 쓰시면서 문의 몇 건이나 자동 처리되는지 보시면,
> 그게 절약한 시간 대비 얼마 가치인지 바로 느끼실 거예요.
> 어떤 부분이 가장 걸리시나요?"

[시간 필요 반응 시]

> "네, 생각해보시고요. 다만 웨지 가격은 3개 클라이언트 한정이라
> 다른 분이 먼저 채우시면 정상 가격(설치비 150만 원)으로 올라가요.
> 언제쯤 결정하실 수 있을 것 같으세요?"
