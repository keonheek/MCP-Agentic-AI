# Korean DR Copy Reference — 화장품 D2C High-Converting Ad Scripts

## 핵심 원칙

한국 화장품 DR 카피는 서양식과 다르다:
- 직접적 주장(claims) 대신 소비자 증언(testimonial) 형식 선호
- "치료" 언어 금지 → 경험/느낌 언어 사용
- 랭킹/수치 신뢰도 높음 ("올리브영 1위", "파우더룸 ★★★★★")
- 성분 교육이 구매 설득에 효과적 (글로우픽 문화)

---

## 훅 패턴 (Hook Patterns)

### 패턴 1: 공감 훅 (Pain Echo)
"[타겟의 정확한 고통을 그대로 묘사]"
예: "겉은 번들거리는데 속은 당기는 피부... 저만 그런 거 아니죠?"

### 패턴 2: 랭킹 신뢰 훅
"[공신력 있는 플랫폼]에서 [기간] 연속 1위"
예: "올리브영에서 2년 연속 1위한 세럼, 드디어 써봤어요"

### 패턴 3: 성분 교육 훅
"[성분명]이 [효과]를 내는 이유, 아무도 안 알려줬죠"
예: "히알루론산이 왜 어떤 건 효과 없는지 아세요?"

### 패턴 4: 비포/애프터 서사
"[사용 전 상황] → [X일 후 변화]"
예: "마스크 떼고 나서 모공이 다 보이던 피부, 3주 후에는..."

### 패턴 5: 반전 훅
"[일반적 통념] 근데 사실은..."
예: "수분크림 많이 바를수록 좋다고요? 이 성분이 없으면 다 의미없어요"

## 광고법 안전 표현 가이드

| ❌ 금지 | ✅ 대체 |
|---|---|
| "피부 재생" | "피부결 개선감", "촉촉한 느낌" |
| "아토피 완화" | "민감한 피부에도 순한 성분" |
| "여드름 치료" | "트러블 케어", "진정 케어" |
| "주름 제거" | "탄력감 UP", "피부결 매끄러움" |
| "미백 효과" | "화사한 피부톤", "밝은 느낌" |
| "임상 완료" (비공인) | "성분 전문가 추천" |
| "OO% 개선" (무증거) | "사용 후 만족하셨다는 후기" |

## 카피 구조 템플릿 (9초 영상)

```
[0-1.5초] 훅 — 타겟의 고통/욕망 정확히 짚기
[1.5-4초] 공감 심화 — "맞죠? 저도 그랬어요"
[4-7초] 솔루션 제시 — 제품 + 핵심 성분 1가지
[7-8.5초] 사회적 증명 — 수치/랭킹/후기 수
[8.5-9초] CTA — "링크에서 확인하세요"
```

---

## 한국 소비자 구매 트리거

1. **랭킹 신뢰**: 올리브영/파우더룸/글로우픽 순위
2. **성분 검증**: 특정 성분에 대한 이해 = 구매 확신
3. **VoC 공명**: "나도 이런 경험 있어" 공감
4. **가격 정당화**: "이 성분이면 이 가격도 납득"
5. **리스크 제거**: 환불보장, 샘플 먼저
