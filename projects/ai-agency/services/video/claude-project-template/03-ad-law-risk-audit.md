# 화장품 광고법 Risk Language Audit Guide

## 법적 근거

- **화장품법 제13조**: 의약품 효능·효과 암시 금지
- **의약외품 기준**: 일부 성분/제품은 의약외품으로 분류 → 별도 심의
- **의료광고심의**: 한의원/병원 광고는 의료광고심의위원회 사전 심의 필수

---

## 절대 금지 표현 (화장품법 위반 → 과태료/광고중단)

### 의약품 효능 암시
- "아토피 완화/치료/개선"
- "여드름 치료", "피부 질환 치료"
- "항균", "살균" (화장품 범주 초과)
- "세포 재생", "DNA 회복"
- "피부 재생" (단, "피부결 개선"은 가능)

### 과대광고
- "OO% 개선" (식약처 인정 임상 없이)
- "세계 최초", "국내 최초" (입증 자료 없이)
- "모든 피부 타입에 효과" (과도한 일반화)
- "부작용 없음" (절대적 표현 금지)

### 의료기기 혼동
- "레이저 효과", "필러 효과" (의료 시술 비교)
- "보톡스급", "시술 대체" 표현

---

## 조건부 허용 표현 (문맥에 따라 다름)

| 표현 | 조건 |
|---|---|
| "임상 시험 완료" | 식약처 인정 기관의 실제 임상 데이터 있을 때만 |
| "피부과 테스트 완료" | 실제 테스트 완료된 경우 |
| "저자극 인증" | 공신력 있는 인증 기관 인증 시 |
| "OO% 함유" | 실제 성분 함량 표시 (효과 주장 아닌 성분 표시) |

---

## 안전한 대체 표현 사전

| 위험 표현 | 안전한 대체 |
|---|---|
| 피부 재생 | 피부결 개선감, 회복되는 느낌 |
| 아토피 완화 | 민감한 피부도 순하게, 진정 케어 |
| 여드름 치료 | 트러블 케어, 피부 진정 |
| 주름 제거 | 탄력감 향상, 매끄러운 피부결 |
| 미백 | 화사한 피부톤, 밝은 느낌 |
| 세포 활성화 | 피부 활력감 |
| OO% 개선 | 사용 후 만족하셨다는 고객 후기 |
| 부작용 없음 | 피부과 테스트 완료 (인증 있을 시) |
| 보톡스급 | (대체 불가 — 사용 금지) |

---

## 한의원 광고 추가 주의사항 (Service 3 한의원 ICP)

- 모든 한의원 광고는 **의료광고심의위원회 사전 심의** 필수
- "치료 효과", "완치" 표현 절대 금지
- 의사/한의사 명시 광고는 면허 번호 표시 권장
- 전후 비교 사진: 실제 환자 동의서 필수

---

## Checklist (광고 발행 전)

- [ ] 의약품 효능 암시 표현 없음
- [ ] 수치(%) 주장에 근거 자료 있음
- [ ] "치료", "재생", "살균" 단어 없음
- [ ] 한의원 광고 시: 심의 번호 확인
- [ ] 비교광고 시: 비교 근거 자료 있음
