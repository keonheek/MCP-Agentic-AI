# LTV:CAC 모델링 템플릿 — 화장품 D2C

## 입력 변수 (클라이언트에게 확인)

| 변수 | 값 | 확인 방법 |
|---|---|---|
| 제품 판매가 | ₩ | 스마트스토어/쇼핑몰 |
| COGS (원가) | ₩ | 클라이언트 직접 확인 |
| 절대 마진 | ₩ | 판매가 - COGS |
| 마진율 | % | 마진/판매가 |
| 월 평균 주문 수 | 건 | 스마트스토어 통계 |
| 재구매율 (12개월) | % | CRM 데이터 (없으면 화장품 평균 35%) |
| 평균 재구매 횟수 | 회 | 재구매율 기반 추정 |

---

## 계산 공식

```
LTV = 판매가 × 평균 재구매 횟수 (12개월)
목표 CAC = LTV × 0.33 (LTV의 1/3 이하)
손익분기 CAC = 절대마진 (1회 구매만으로 이익)

광고 효율 기준:
- CAC < 손익분기 CAC: 즉시 흑자
- CAC < 목표 CAC: 건강한 사업
- CAC > LTV: 광고 중단 필요
```

---

## 샘플 계산 (토리든 기준)

| 항목 | 수치 |
|---|---|
| 제품 판매가 | ₩32,000 |
| COGS | ₩8,000 |
| 절대 마진 | ₩24,000 |
| 마진율 | 75% |
| 재구매율 (추정) | 40% |
| 평균 재구매 횟수 | 1.8회 |
| LTV (12개월) | ₩57,600 |
| 목표 CAC | ₩19,200 (LTV × 0.33) |
| 손익분기 CAC | ₩24,000 |

**판단:** CAC ₩19,200 이하면 12개월 기준 건강. 
월 광고비 ₩500k, 목표 CAC ₩19,200 → 월 26건 이상 전환 필요.

## 광고 예산 시나리오

| 월 광고비 | 목표 CAC | 필요 전환 수 | 필요 ROAS |
|---|---|---|---|
| ₩300,000 | ₩19,200 | 16건 | 1.25x |
| ₩500,000 | ₩19,200 | 26건 | 1.25x |
| ₩1,000,000 | ₩19,200 | 52건 | 1.25x |

## 자격검증 판단 기준

**GREEN (진행):** 절대 마진 ₩30,000+ AND COGS의 3배 이상
**YELLOW (주의):** 마진율 50%+ 하지만 절대 마진 ₩20,000-30,000
**RED (거절):** 절대 마진 ₩20,000 미만 OR 마진율 50% 미만
