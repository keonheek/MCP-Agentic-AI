# Hormozi Grand Slam Offer Framework — 한국어 요약

## 핵심 개념

"최고의 오퍼는 고객이 '이걸 안 사면 바보'라고 느끼게 만드는 것."

광고가 아무리 좋아도 오퍼가 약하면 안 팔린다. 광고 대행 전 오퍼 강도 먼저 점검.

---

## 5가지 오퍼 레버

### 1. 가치 (Dream Outcome)
고객이 얻는 최종 결과가 명확한가?
- 약: "좋은 피부"
- 강: "3주 안에 모공이 보이지 않는 피부 — 돈 환불 보장"

점수 기준:
- 10점: 꿈의 결과가 구체적이고 측정 가능
- 5점: 결과는 있지만 막연함
- 1점: "좋아진다"는 수준

### 2. 가능성 (Perceived Likelihood)
고객이 "나도 될 것 같다"고 믿는가?
- 증거: 후기 수, 실제 사용 전후, 성분 데이터
- 사회적 증명: "올리브영 1위", "후기 10만개"

### 3. 노력 (Effort & Sacrifice)
사용이 쉬운가? 루틴에 얼마나 추가되는가?
- 약: "매일 3단계 루틴 추가 필요"
- 강: "기존 루틴 3단계 중 1개만 교체"

### 4. 시간 (Time to Result)
얼마나 빨리 효과를 느끼는가?
- 한국 화장품 구매자 평균 인내심: 2-4주
- 1주 내 변화 느낌 주장이 가장 강력

### 5. 위험 (Risk Reversal)
안 사면 손해 없는가?
- 최강: 100% 환불 보장
- 중간: 30일 교환 보장
- 약: "만족 못 하면 환불" (조건 불명확)

## 오퍼 강화 빠른 레시피 (화장품)

1. **보장 추가**: "14일 내 차이 없으면 전액 환불" → Risk 10점
2. **번들링**: 세럼 + 미스트 패키지로 가치 UP → Dream Outcome 강화
3. **사회적 증명**: 파우더룸/글로우픽 리뷰 수 전면 배치 → Likelihood UP
4. **속도 강조**: "첫 주 사용 후기" 기반 카피 → Time UP
5. **사용법 단순화**: "1단계 교체" 메시지 → Effort DOWN
