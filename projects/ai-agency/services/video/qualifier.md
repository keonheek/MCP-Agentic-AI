# 고객 자격검증 체크리스트 (Lead Qualifier)

_Service 1: AI 영상 광고. 모든 항목 통과 시에만 진행._
_체크 실패 → "핏이 아닙니다" 정중히 거절. 안 받는 고객 정책 = 모트의 일부._

---

## 필수 통과 항목 (Hard Gates — 하나라도 실패 시 즉시 거절)

### 1. 마진 구조
- [ ] 제품 절대 마진 **₩30,000+** per unit
- [ ] 마진이 **COGS의 3배 이상** (둘 다 충족해야 함)
- [ ] 예: COGS ₩10,000 → 마진 ₩30,000+ (판매가 ₩40,000+)

> 이유: 광고비 ₩500k/월 쓰면 LTV:CAC 맞추려면 마진이 충분해야 함.

### 2. 제품 물류
- [ ] 신발박스 안에 들어가는 크기
- [ ] 상온 보관 가능 (냉장/냉동 X)

> 이유: 반품/환불 비용이 마진을 잡아먹으면 광고 ROI 불가.

### 3. 시장 적합성
- [ ] **고관여 시장**의 문제를 해결하는가 (화장품, 건기식, 한의원 등)
- [ ] 구매 결정에 감정적 요소가 있는가 (공포, 욕망, 부끄러움 등)

### 4. 오퍼 강도 (Hormozi 5요소) — 7점 이상
| 요소 | 점수 (1-10) | 질문 |
|---|---|---|
| 가치 | | "고객이 얻는 결과가 가격 대비 압도적인가?" |
| 가능성 | | "실제로 효과 있다고 믿을 수 있는가?" |
| 노력 | | "사용이 쉬운가?" |
| 시간 | | "효과가 빠른가?" |
| 위험 | | "환불/보증 정책이 있는가?" |

> 7점 미만 오퍼는 광고가 아무리 좋아도 안 팜.

### 5. 광고 예산
- [ ] 광고 집행 예산 **₩500,000+/월** 확보 가능
- [ ] Meta/Instagram 또는 Naver 광고 집행 경험 (없어도 되지만 각오 있어야 함)

### 6. 디지털 존재감
- [ ] 아래 중 **2개 이상** 보유:
  - [ ] 사업자등록 도메인 + 웹사이트
  - [ ] Instagram 계정 (개설 기준, 팔로워 무관)
  - [ ] 네이버 스마트스토어

### 7. 제품 이미지
- [ ] **화이트 배경 제품 PNG** 최소 1장 보유
- [ ] 해상도 1080px 이상

## 추가 확인 항목 (Soft Checks — 없으면 진행 가능하나 주의)

- [ ] 기존 고객 리뷰 10개 이상 (VoC 수집 용이성)
- [ ] Meta Ad Library에 경쟁사 광고 존재 (시장에 돈이 흐름을 의미)
- [ ] 카카오톡 채널 보유 (Service 2 업셀 가능성)
- [ ] 재구매율 데이터 있음 (LTV 계산 가능)

---

## 자격검증 5문항 (Discovery Call 전 카톡/이메일로 먼저 확인)

> 이 질문으로 30초 안에 하드 게이트 통과 여부 파악.

1. 제품 판매가와 원가(COGS)가 어떻게 되세요?
2. 현재 월 광고 예산이 어느 정도 되세요?
3. 인스타그램 계정이나 스마트스토어 운영 중이신가요?
4. 화이트 배경 제품 사진 있으신가요?
5. 지금 가장 큰 마케팅 고민이 뭐세요?
