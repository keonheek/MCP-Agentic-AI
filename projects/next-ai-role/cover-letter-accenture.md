# Accenture Song / Global AI & Data Development Program — 자기소개서

안녕하세요.

저는 성균관대학교 경영학부 재학 중인 김건희입니다. 저는 AI 기술을 단순히 이해하는 것을 넘어, 기업 클라이언트에게 즉시 가치를 제공하는 배포 가능한 시스템을 직접 설계하고 구축하는 경험을 쌓아왔습니다.

**FinAgent(keonhee-finagent.streamlit.app)는 제 역량의 가장 좋은 증거입니다.**

FastAPI 백엔드, 커스텀 VectorDB, Text2SQL 엔진, 그리고 Streamlit 프론트엔드로 구성된 프로덕션급 시스템입니다. 사용자는 간단한 자연어 질문("삼성전자 2024년 영업이익이 얼마인가?")을 입력하면, 시스템은 자동으로 재무 데이터를 조회하고 분석하여 답변합니다.

이 프로젝트는 문제 정의(금융사의 분석 효율성 문제) → 기술 선정(LangGraph, RAG, Text2SQL) → 시스템 설계 → 배포(Streamlit Cloud)까지 **클라이언트 제공 가능한 완전한 솔루션**임을 보여줍니다. Accenture Song이 디지털 트랜스포메이션 프로젝트에서 요구하는 end-to-end 역량입니다.

추가로, 저는 Notion, GitHub 등 엔터프라이즈 도구와 연결되는 **커스텀 MCP 서버 개발 경험**이 있습니다. Accenture가 2026년 MCP 기반 AI 워크플로우로 전환하는 과정에서 즉시 기여할 수 있습니다.

저는 기술 깊이와 비즈니스 임팩트를 동시에 추구하는 인재입니다.

인터뷰 기회를 요청드립니다.
