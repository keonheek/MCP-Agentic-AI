# BCG Gamma (BCGX) 커버레터

---

BCG Gamma 팀 여러분께,

저는 SKKU 경영학과 학생 김건희입니다. 금융 분석 시스템을 프로덕션 수준으로 설계·배포한 경험을 바탕으로 지원합니다.

**FinAgent** (keonhee-finagent.streamlit.app)는 다중 에이전트 금융 분석 시스템으로, 사용자의 자연어 질문을 자동으로 SQL 쿼리로 변환하고 RAG를 통해 재무 맥락을 추가해 분석 결과를 생성합니다. 처음에는 ChromaDB를 사용했으나 검색 정확도 문제가 발생했고, 제약 조건 하에서 OpenAI 임베딩 + 코사인 유사도 기반의 맞춤형 벡터DB를 구축했습니다. 이는 단순히 기술 선택이 아니라 데이터 특성을 분석 → 아키텍처 결정 → 구현 → 배포라는 구조화된 문제 해결 과정이었습니다.

이러한 설계 결정 능력 — 제약 조건을 마주했을 때 근본적인 데이터 특성부터 파악해 프로덕션 솔루션을 만드는 능력 — 이것이 BCG Gamma가 원하는 것이라고 믿습니다. 저는 기술 스택 자체보다는, 데이터에서 비즈니스 통찰력을 추출하고 실제로 동작하는 시스템으로 만들어내는 문제 해결 방식에 집중합니다.

저는 경영학과 AI 기술을 동시에 이해하는 드문 인재입니다. BCG Gamma 서울 오피스에서 인터뷰 기회를 얻을 수 있기를 기대합니다.
