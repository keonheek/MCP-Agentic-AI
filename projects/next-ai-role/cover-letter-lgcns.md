**FinAgent** (keonhee-finagent.streamlit.app)는 LangGraph 기반 다중 에이전트 금융 분석 시스템입니다. Router Agent가 자연어 질문을 분석해 Text2SQL 또는 RAG 에이전트로 동적 라우팅하고, 최종적으로 구조화된 분석 리포트를 생성합니다. ChromaDB가 Python 3.14와 호환되지 않는 상황에서 외부 의존성 없이 OpenAI 임베딩 + NumPy 코사인 유사도 기반 커스텀 VectorDB를 직접 구현해 배포 환경의 제약을 극복했습니다.

**DART MCP Server**는 금융감독원 DART 시스템의 기업 재무 데이터를 AI 에이전트가 직접 조회할 수 있는 커스텀 서버입니다. KOSPI/KOSDAQ 상장 2,500개+ 기업의 재무제표, 공시를 AI 도구로 노출하며, LG전자·LG화학 등 LG 계열사 재무 분석에도 직접 활용 가능합니다.

LG CNS는 국내 최대 B2B IT 서비스 기업으로, AI 솔루션을 실제 기업 환경에 구현하는 역할이 제가 쌓아온 역량과 직접 연결됩니다. 경영학 전공을 통한 비즈니스 이해와 end-to-end AI 기술 구현 경험을 결합해 LG CNS의 AI 프로젝트에 기여하고 싶습니다.
