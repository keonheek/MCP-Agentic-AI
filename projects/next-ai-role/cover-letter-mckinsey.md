# McKinsey QuantumBlack 커버레터

---

McKinsey QuantumBlack 팀 여러분께,

저는 SKKU 경영학과 학생 김건희입니다. Python 기반의 엔드-투-엔드 데이터 파이프라인을 설계·배포한 경험을 바탕으로 지원합니다.

**FinAgent** (keonhee-finagent.streamlit.app)는 LangGraph + Text2SQL + RAG를 통합한 금융 분석 시스템입니다. 자연어 쿼리를 SQLite 데이터베이스에 대한 동적 SQL로 변환하고, pandas를 활용한 데이터 전처리와 RAG 파이프라인을 거쳐 최종 분석 결과를 생성합니다. 전체 로직은 Python으로 구현되었고, FastAPI를 통해 백엔드를 구축했으며 Streamlit으로 프로덕션에 배포했습니다.

또한 **DART MCP Server**를 개발했습니다. 이는 한국 금융 데이터(삼성전자, SK하이닉스 등)에 접근하는 맞춤형 도구로, 자사 API를 직접 통합해 구조화된 재무 데이터를 추출하는 데이터 인프라입니다.

이 두 프로젝트는 데이터 수집 → SQL 파이프라인 → 통계 분석 → 비즈니스 추천이라는 QuantumBlack의 핵심 워크플로우를 그대로 구현한 것입니다. 저는 경영학적 직관과 기술적 실행 능력을 동시에 갖춘 인재입니다.

McKinsey 서울 오피스에서 인터뷰 기회를 얻을 수 있기를 기대합니다.
