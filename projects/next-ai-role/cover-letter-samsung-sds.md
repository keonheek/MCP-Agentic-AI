Samsung SDS AI 솔루션 팀 담당자님께,

저는 성균관대학교 경영학부 재학 중인 김건희입니다. 프로덕션 수준의 에이전트 AI 시스템을 설계하고 배포한 경험을 바탕으로 지원합니다.

**FinAgent** (keonhee-finagent.streamlit.app)는 LangGraph 기반 다중 에이전트 금융 분석 시스템입니다. 자연어 질의 → Text2SQL → SQLite 조회 → RAG 문서 검색 → 분석 리포트 생성까지 end-to-end로 동작하며, 현재 배포 운영 중입니다. ChromaDB가 Python 3.14와 호환되지 않는 상황에서 외부 의존성 없이 OpenAI 임베딩 + NumPy 코사인 유사도 기반 커스텀 VectorDB를 직접 구현해 문제를 해결했습니다.

**DART MCP Server**는 한국 금융감독원 DART 시스템의 기업 재무 데이터를 AI 도구로 직접 노출하는 커스텀 MCP 서버입니다. KOSPI/KOSDAQ 상장 2,500개+ 기업의 재무제표와 공시를 Claude Code에서 자연어로 조회할 수 있습니다. 삼성전자, 현대차 등 한국 대기업의 데이터 파이프라인에 대한 실무 이해를 갖추고 있습니다.

Samsung SDS는 국내 최대 규모의 IT 서비스 기업으로, AI 솔루션을 실제 기업 운영에 적용하는 방식이 제가 쌓아온 역량과 직접적으로 연결됩니다. 경영학 전공을 통한 비즈니스 이해와 AI 기술 구현 경험을 결합해 Samsung SDS의 엔터프라이즈 AI 프로젝트에 기여하고 싶습니다.
