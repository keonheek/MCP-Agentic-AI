## 제7장: 소비자, 생산자, 시장의 효율성

### 후생경제학(Welfare Economics)

**후생경제학**은 자원배분이 경제적 후생(사회 전체 이익)에 미치는 영향을 분석하는 분야다.

### 소비자잉여(Consumer Surplus)

#### 정의

**소비자잉여** = 소비자의 **지불용의(Willingness to Pay)** - 실제 지불 가격

**지불용의**: 소비자가 어떤 재화를 구매하기 위해 기꺼이 지불하려는 최고 금액

#### 그래프에서의 의미

- 수요곡선 아래, 시장가격 위의 영역이 소비자잉여
- **소비자잉여가 크다** = 구입자들이 매우 만족한다

#### 가격 인하 시 변화

- 가격 ↓ → 소비자잉여 ↑ (더 큰 영역)
- 가격 ↑ → 소비자잉여 ↓ (더 작은 영역)

### 생산자잉여(Producer Surplus)

#### 정의

**생산자잉여** = 실제 판매가격 - **공급비용(생산에 드는 비용)**

생산자잉여는 판매자의 **경제적 이윤(Economic Profit)**과 같다.

#### 그래프에서의 의미

- 시장가격 위, 공급곡선 아래의 영역이 생산자잉여
- **생산자잉여가 크다** = 판매자들이 높은 이윤을 거둔다

#### 주의: 경제적 이윤 ≠ 회계이윤

- **경제적 이윤**: 판매가격 - 경제적 비용 (기회비용 포함)
- **회계이윤**: 판매가격 - 회계적 비용 (명시적 비용만)

### 총잉여(Total Surplus)

#### 정의

**총잉여 = 소비자잉여 + 생산자잉여**

또는

**총잉여 = 지불용의 - 공급비용**

이는 사회 전체의 경제적 후생을 나타낸다.

### 효율성과 형평성

#### 효율성(Efficiency)

- **정의**: 총잉여를 극대화하는 자원배분 상태
- "가장 효율적" = 사회 전체의 경제적 이익이 최대

#### 형평성(Equity)

- **정의**: 총잉여(또는 자원)의 분배의 공정성 정도
- 효율성과 형평성은 서로 독립적

#### 관계

- 효율적이지만 불공정한 배분 가능 (예: 극소수가 대부분의 부를 가짐)
- 공정하지만 비효율적인 배분 가능 (예: 강제적으로 평등하게 나누면 생산 의욕 저하)

### 자유시장의 균형과 효율성

#### 핵심 결론

**자유시장 경쟁 균형은 총잉여를 극대화한다** = 효율적 자원배분

이유:
- 시장은 모든 거래이득을 실현함
- 추가 거래로 한쪽이 더 나아지고 다른 쪽이 나빠질 수 없는 상태 (파레토 효율)

### 시장실패(Market Failure)

시장이 효율적 자원배분을 실현하지 못하는 경우:

1. **시장지배력(Market Power)**: 한 기업이 가격에 영향력을 행사 (독점 등)
2. **외부효과(Externality)**: 한 사람의 행동이 제3자에게 비자발적 영향
   - 양의 외부효과: 사회에 유익 (예: 기술 개발)
   - 음의 외부효과: 사회에 해로움 (예: 공해)

### 함정 포인트

- **지불용의 ≠ 실제 가격**: 지불용의가 실제 가격보다 크면 소비자는 구매
- **소비자잉여 ≠ 소비 만족도**: 소비자잉여는 금전적 개념, 주관적 만족도와는 다름
- **생산자잉여 = 경제적 이윤**: 회계이윤과 혼동하지 말 것
- **효율성 ≠ 형평성**: 효율적이어도 불공정할 수 있고, 그 역도 참
- **시장실패의 2가지 원인**: 시장지배력과 외부효과만 언급 (정보비대칭, 공공재 등도 있지만 이 장에서는 아님)
- **균형 = 효율적**: 자유시장 경합 균형은 총잉여를 극대화함

## 제8장: 조세의 경제적 비용

### 세금 부과의 효과

세금을 부과하면:

| 항목 | 변화 |
|------|------|
| 소비자잉여 | ↓↓ |
| 생산자잉여 | ↓↓ |
| 정부 조세수입 | = T × Q (T: 세금, Q: 거래량) |
| **총잉여** | ↓ (손실 발생) |

### 경제적 순손실(Deadweight Loss, DWL)

#### 정의

**경제적 순손실** = 세금 부과 후 총잉여의 순감소

**경제적 순손실 = 원래 총잉여 - 세금 후 총잉여**

또는

**경제적 순손실 = (소비자잉여↓ + 생산자잉여↓) - 조세수입**

#### 원인

세금이 거래량을 감소시키기 때문:
- 거래량 ↓ → 양쪽이 이득을 보지 못함 (죽은 무게)
- 정부는 조세수입을 얻지만, 그보다 더 많은 잉여가 사라짐

#### 그래프에서의 의미

- 경제적 순손실 = 수요곡선과 공급곡선 사이의 삼각형 영역 (거래량이 줄어든 부분)

### 세금과 경제적 순손실의 관계

**경제적 순손실 ∝ (세금)²**

- 세금이 2배 증가 → 경제적 순손실은 4배 증가 (비선형 관계)
- 이는 세금이 높을수록 왜곡이 심해지는지를 설명

### 탄력성과 경제적 순손실

**탄력성이 클수록 경제적 순손실이 크다**

이유:
- 탄력적 → 가격 변화에 민감 → 거래량이 크게 감소 → 손실이 큼
- 비탄력적 → 가격 변화에 둔감 → 거래량이 적게 감소 → 손실이 작음

예:
- 담배세: 수요가 비탄력적 → 손실 작음, 세수 많음 (좋은 세금)
- 사치재세: 수요가 탄력적 → 손실 큼, 세수 적음 (나쁜 세금)

### 레퍼 곡선(Laffer Curve)

**레퍼 곡선**은 세율과 조세수입의 관계를 보여준다.

#### 특성

- 세율 0% → 조세수입 0
- 세율 증가 → 조세수입 증가 (처음엔)
- **어떤 지점 이후** → 세율 증가 → 조세수입 감소 (왜곡이 심해짐)
- 세율 100% → 조세수입 0 (아무도 일하지 않음)

#### 함의

- 최적 세율이 존재 (조세수입 극대화)
- 세율이 너무 높으면 비효율 심화
- 실제 경제에서 "조세수입이 최대인 세율"을 어떻게 찾을 것인가가 과제

### 함정 포인트

- **조세수입 ≠ 국가이익**: 조세수입 내에는 경제적 순손실이 내재됨
- **경제적 순손실은 피할 수 없음**: 세금이 존재하면 반드시 손실 발생
- **경제적 순손실 계산**: 세금액 × 거래량 변화의 절반 정도 (정확하지 않지만 직관적 계산)
- **세금 크기 vs 손실**: 비선형 관계 (세금 2배 → 손실 4배)
- **탄력성 vs 손실**: 탄력성 크면 손실도 크다 → 탄력적 시장에는 세금 부과 피하기
- **레퍼 곡선의 실제 형태**: 이론적 개념이며, 실제 경제의 최적세율 위치는 경험적으로 결정

## 제9장: 국제무역

### 국제 가격과 비교우위

#### 국제가격(World Price) vs 국내가격(Domestic Price)

- **국제가격**: 국제시장에서의 가격
- **국내가격**: 무역이 없을 때 국내 시장의 균형가격

#### 소규모개방경제(Small Open Economy)

**소규모개방경제**: 국제가격에 영향을 미치지 못하는 개별 국가

- 한 국가가 아무리 생산을 늘려도 국제가격은 변하지 않음
- 각 국가는 국제가격을 받아들이는 **가격수용자(Price-Taker)**

#### 비교우위와 무역 방향 결정

- **국내가격 < 국제가격** → 이 재화에 **비교우위** → **수출국**
- **국내가격 > 국제가격** → 이 재화에 **비교우위 없음** → **수입국**

이유: 국내가격이 낮다 = 생산비용이 낮다 = 비교우위가 있다

### 수출국(Exporting Country)의 변화

국제가격 > 국내가격일 때, 무역 개시:

| 항목 | 변화 | 이유 |
|------|------|------|
| 국내가격 | ↑ | 국제가격 쪽으로 수렴 |
| 국내생산 | ↑ | 높은 가격으로 더 많이 생산 |
| 국내소비 | ↓ | 높은 가격으로 덜 소비 |
| 국내수요량 - 국내공급량 | < 0 (음수) | 초과공급 = 수출 |
| **소비자잉여** | **↓↓** | 가격 상승으로 손실 |
| **생산자잉여** | **↑↑↑** | 가격 상승으로 크게 이득 |
| **총잉여** | **↑** | 전체적으로 이득 |

#### 결론

- **소비자는 피해** (가격 올라 소비 줄음)
- **생산자는 크게 이득** (더 높은 가격에 판매)
- **국가 전체는 이득** (총잉여 증가)

### 수입국(Importing Country)의 변화

국제가격 < 국내가격일 때, 무역 개시:

| 항목 | 변화 | 이유 |
|------|------|------|
| 국내가격 | ↓ | 국제가격 쪽으로 수렴 |
| 국내생산 | ↓ | 낮은 가격으로 덜 생산 |
| 국내소비 | ↑ | 낮은 가격으로 더 소비 |
| 국내수요량 - 국내공급량 | > 0 (양수) | 초과수요 = 수입 |
| **소비자잉여** | **↑↑↑** | 가격 하락으로 크게 이득 |
| **생산자잉여** | **↓↓** | 가격 하락으로 손실 |
| **총잉여** | **↑** | 전체적으로 이득 |

#### 결론

- **소비자는 이득** (가격 내려 소비 늘림)
- **생산자는 피해** (더 낮은 가격에 판매)
- **국가 전체는 이득** (총잉여 증가)

### 관세(Tariff)

#### 정의

**관세**: 수입품에 부과하는 세금

#### 관세의 효과

관세 부과 → 수입품의 국내가격 ↑ (관세만큼)

결과:

| 항목 | 변화 | 방향 |
|------|------|------|
| 국내가격 | ↑ | 국제가격에 가까워짐 |
| 국내생산 | ↑ | 높은 가격으로 더 생산 |
| 국내소비 | ↓ | 높은 가격으로 덜 소비 |
| 수입량 | ↓↓ | 크게 감소 |
| **소비자잉여** | **↓** | 가격 상승으로 손실 |
| **생산자잉여** | **↑** | 가격 상승으로 이득 |
| **정부 관세수입** | **↑** | 관세 × 수입량 |
| **총잉여** | **↓** | **경제적 순손실 발생** |

#### 관세의 비효율성

- 소비자 손실 > 생산자 이득 + 정부 수입
- 결과: 국가 전체의 경제적 순손실
- 무역제한은 총잉여를 감소시킨다 (비효율적)

### 무역제한 옹호 논리

무역이 효율적이라는 것을 알면서도, 일부가 무역제한을 주장하는 이유:

1. **일자리 상실 우려**
   - 수입품 증가 → 국내 산업 일자리 감소
   - 하지만: 다른 산업에서 새로운 일자리 창출 가능 (장기)

2. **국가안보(National Security)**
   - 전략적으로 중요한 산업 보호 필요 (예: 식량, 에너지)
   - 예: 식량자급률 유지, 무기 생산 기지 유지

3. **유치산업 보호(Infant Industry)**
   - 발전도상국의 신생 산업이 성장할 때까지 보호
   - 성숙하면 보호 해제 예정 (하지만 실제로는 오래 보호됨)

4. **불공정 경쟁 대응**
   - 상대국이 덤핑(정상가격 이하 판매) 또는 부정행위 → 반제재

5. **협상 전략**
   - 상대국의 관세 인하를 유도하기 위한 협상 카드
   - 예: "너희 관세 내리면 우리도 내린다"

### 함정 포인트

- **국내가격 vs 국제가격 비교**: 비교우위 판단의 핵심
- **수출국의 소비자 피해**: 무역이 국가 전체에는 이득이지만 특정 그룹(소비자)은 피해
- **수입국의 생산자 피해**: 마찬가지로 생산자가 피해를 봄
- **관세의 경제적 순손실**: 조세처럼 경제적 순손실 발생 (피할 수 없음)
- **총잉여 ≠ 모두의 이익**: 무역으로 총잉여가 증가해도 일부 피해자 발생 (분배 문제)
- **유치산업 보호의 역설**: 보호되던 산업이 계속 보호를 요청하는 경향 (정치경제학 문제)
- **국가 전체의 이득 ≠ 개인의 이득**: 경제학은 효율성을 강조하지만, 사회는 분배도 고려해야 함
