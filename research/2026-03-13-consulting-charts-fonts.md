# Research: Consulting Presentation Standards — Fonts & Charts
Date: 2026-03-13

---

## Summary

- **Fonts:** McKinsey uses **Calibri (sans-serif)** for global English documents; consulting firms are not standardized across the industry, but trend toward clean sans-serif for modern readability. Specific font choices for BCG, Bain, Deloitte, Accenture are proprietary/not publicly disclosed.
- **Font hierarchy:** Consulting slides enforce strict sizing: titles 28-44pt (max 2 lines), body 18-24pt, subtext 14-18pt. Max 2 font families per deck.
- **Charts for trend analysis:** Line charts (preferred) or column charts for revenue/profit over time.
- **Charts for margin & comparison:** Stacked bar/column charts for component breakdown; regular bar charts for competitor/segment comparison.
- **DCF & waterfall:** Waterfall charts show how initial values increase/decrease through series of changes — ideal for DCF component breakdowns; sensitivity analysis uses scatter plots or heatmaps.
- **Consulting-quality standard:** Charts must have one message per chart, source labels, minimal ink (no gridlines/3D effects), strategic color accents only (green for positive, brand colors for emphasis), full axis labels, and strategic data callouts.
- **Color palette:** 3-4 colors max per deck; green for positive metrics, gray for secondary info, brand color primary.

## Detailed Findings

### 1. Typography & Font Standards

#### McKinsey (Confirmed)
- **Font:** Calibri (sans-serif) for global English contracts and client-facing documents
- **Regional variation:** Korean versions use Batangche (serif) for formal tone
- **Rationale:** Calibri aligns with trend toward clean, sans-serif fonts for digital and global use

#### BCG, Bain, Deloitte, Accenture
- No public specification of exact fonts in client presentations
- Brand guidelines are proprietary/internal
- Industry trend: **Sans-serif fonts dominate** (cleaner, more modern than serif)
- Standard across MBB firms: Max 2 font families per presentation, consistent throughout

#### Font Size Hierarchy (Mandatory for All Consulting Firms)

| Element          | Size Range      | Rules                           |
|------------------|-----------------|---------------------------------|
| **Slide Titles** | 28-44pt         | Bold, max 2 lines, action title (full sentence stating takeaway) |
| **Body Text**    | 18-24pt         | Left-aligned, consistent within slide |
| **Sub-bullets**  | 14-18pt         | Gray text for secondary information |

**Key rule:** Slides readable in 60 seconds from distance; title guides eye and states key insight upfront (Pyramid Principle).

### 2. Chart Types for Financial Analysis

#### Revenue & Profit Trend Analysis
- **Primary choice:** Line charts (connect data points, highlight long-term trends)
- **Secondary choice:** Column charts (vertical bars, intuitive for time-series)
- **When to use area charts:** To show both trend AND magnitude of change (e.g., revenue growth % + absolute volume)

#### DCF Valuation & Component Breakdown
- **Chart type:** Waterfall charts
- **Structure:** Shows initial value → positive/negative incremental changes → final value
- **Use case:** Break down DCF valuation into drivers (EBIT, tax rate, WACC, terminal value, etc.)
- **Best practice:** Label each segment with percentage or absolute value; use green for value increases, red/gray for decreases

#### Sensitivity Analysis (DCF assumptions)
- **Scatter plots:** Explore relationships between assumptions (e.g., WACC vs. valuation outcome)
- **Heatmaps:** Show sensitivity of valuation to two variables (e.g., revenue growth % vs. EBITDA margin)
- **Tornado charts:** Rank assumptions by impact (which drives biggest change to valuation)

#### Margin Analysis by Segment/Competitor
- **Stacked bar/column charts:** Show component contribution to total (e.g., gross margin → OpEx → EBIT margin)
- **Grouped bar charts:** Compare margins across competitors side-by-side
- **Best practice:** Sort from largest to smallest (except time-series); group by logical category

#### Company Comparison (M&A due diligence context)
- **Grouped column charts:** Compare key metrics (revenue, EBITDA, margin) across target + peers
- **Waterfall with comparison:** Show value bridge (EBITDA to valuation) with peer benchmarks on side
- **Scatter plots:** Position target on bubble chart (x-axis = growth, y-axis = EBITDA margin, bubble size = revenue)

### 3. Consulting-Quality Chart Standards

#### What Makes a Consulting Chart Different from Excel Default

| Dimension       | Standard Excel                     | Consulting-Quality                  |
|-----------------|------------------------------------|------------------------------------|
| **Message**     | Data dump (auto-formatted)         | Single takeaway per chart (e.g., "EBITDA grew 15% YoY due to margin expansion") |
| **Design**      | Cluttered grids, 3D effects, multiple colors everywhere | Clean lines, 2D only, white space, strategic color use |
| **Ink reduction** | All gridlines, legends on chart   | Minimal ink: remove gridlines unless critical, legend off-slide |
| **Labels**      | Auto-labeled axes, no sources      | Full axis labels, data callouts on key points, source citation |
| **Complexity**  | Overloaded (all data on one chart) | Filtered to essentials; dense data relegated to dashboards |
| **Polish**      | Default font/spacing               | Aligned to slide grid, consistent hierarchy, professional fonts |
| **Color**       | Default color scheme               | 3-4 colors max; brand color primary, green for positive metrics, gray for secondary |

#### Consulting Chart Checklist
- [ ] **One message per chart** — Title is a full sentence (not "Revenue by Region")
- [ ] **Source labeled** — "Source: [Financial statements, Industry report, etc.]"
- [ ] **Strategic color accents** — Green for positive change, red for negative, brand color for emphasis
- [ ] **Data callouts** — Key numbers (15%, $5M, etc.) labeled directly on bars/points, not legend
- [ ] **No 3D or decoration** — Every visual element serves the message
- [ ] **Readable in 60 seconds** — Minimal text, clear hierarchy, intuitive reading path
- [ ] **Grid-aligned on slide** — Consistent with deck template and margins

### 4. Consulting Slide Deck Structure (McKinsey/BCG Standard)

#### Required Slide Elements
1. **Title slide** — Company name, engagement title, date
2. **Executive summary** — Recommendation/answer first, then key supporting arguments (Pyramid Principle)
3. **Content slides** — One point per slide, supporting evidence, charts with messages
4. **Conclusion/next steps** — Decisions, actions, timeline
5. **Appendix** — Detailed data, methodologies, source notes

#### Mandatory Formatting Rules (All MBB Firms)
- **Max 2 font families** per deck
- **Max 3-4 colors** in color palette (brand + accent + gray)
- **Titles are action statements** — "Revenue growth driven by 3 factors" (not "Revenue Growth")
- **MECE structure** — Mutually exclusive, collectively exhaustive (no overlap, no gaps)
- **Alignment grid** — All elements snap to grid; white space consistent
- **One slide per point** — No multi-topic slides
- **Consistent margins** — Fixed title placement, footer with slide number/date

#### Font Size Example Template
- Title: 36pt bold (Calibri or Arial)
- Body bullet: 20pt regular
- Sub-bullet: 16pt gray
- Chart labels: 14pt (on chart), 18pt (axis labels)

---

### 5. Color Palette Standards

#### McKinsey Example
- Primary: Deep blue (brand)
- Secondary: Light gray (supporting info)
- Accent: Green (positive metrics), light red (warning/negative)
- Background: White (no patterns)

#### General Consulting Palette Rules
- **3-4 colors max** per slide
- **High contrast** — Ensure text readable against backgrounds
- **Consistency** — Same colors used for same metrics across all slides (green always = positive)
- **No decoration** — Color only denotes data importance, never aesthetic flair
- **Print-safe** — Color scheme must survive B&W printing for executive distribution
